【游戏名称】
雪人游戏

【游戏介绍】
这是一款基于Phaser引擎开发的收集类小游戏。游戏中，你扮演一只可爱的熊猫，在美丽的樱花花园中收集从天而降的物品。你要做的就是收集好物品获得高分，避开危险物品保护生命值，坚持得分更高！

【玩法说明】
游戏中物品会从屏幕上方不断掉落，熊猫需要左右移动接住好物品，躲避坏物品。

道具：
路上会不定时掉落樱花花瓣、金币、蝴蝶、枯叶、炸弹。

1. 粉色樱花花瓣：+10分，最常见的好物品。
2. 金色金币：+20分，高分物品，优先收集。
3. 紫色蝴蝶：+15分，中等分数物品。
4. 棕色枯叶：-10分且失去1条生命，双重惩罚的"潜伏杀手"。
5. 红色炸弹X：失去1条生命，会脉动闪烁警告。

操作：
你可以通过鼠标移动或手指拖动，熊猫会跟随你的位置左右移动。
也可以使用键盘左右方向键控制熊猫移动。

目标：
你有5条生命，碰到枯叶或炸弹会失去生命。生命耗尽就会游戏结束！

Tips：游戏结束会显示你的最终分数。点击重新开始，刷新你的最高分记录！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Phaser 3.60.0游戏引擎，Arcade物理引擎，Canvas 2D渲染技术，支持PC和移动端浏览器。
